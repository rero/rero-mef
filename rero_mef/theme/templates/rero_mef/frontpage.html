MEF (Multilingual Entity File)

Version v{{ version }}

MEF is a prototype server under development, with records for persons, organisations, works, expressions, places, time spans and common names for reuse in integrated library systems (ILS). The metadata model relies on various library entity files, creating multilingualism in the data. Alignments are realized automatically based on existing relations.

API entrypoints:

  • agents
  • concepts
  • places
